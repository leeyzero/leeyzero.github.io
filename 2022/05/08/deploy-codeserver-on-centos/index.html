CentOS搭建CodeServer环境

Posted on 2022-05-08 Edited on 2024-10-20 In 开发工具

Code Server是一个基于VSCode实现的开源的云端IDE，只要能联网，就可以通过浏览器进行访问，无需安装，十分方便。本文主要介绍如何在CentOS 8中安装Code Server。

依赖环境

  • 2GB RAM
  • 需要root权限，对于非root账号，需要sudo权限
  • 需要安装nginx
